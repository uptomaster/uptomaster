### 👋 About Me

기능을 구현하는 것에서 끝나지 않고,  
그 기능이 **운영 환경에서 어떻게 동작하고 유지되는지**를 함께 고민하며  
프로젝트를 진행해 왔습니다.

웹 서비스 개발을 시작으로  
서버 · 데이터베이스 연동,  
리눅스 기반 환경 구성,  
네트워크 구조 설계와 보안 점검까지 경험하며  

단순 구현이 아닌  
**운영 관점의 시스템 설계와 신뢰성**을 목표로 성장하고 있습니다.
