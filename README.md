## 🧭 About Me

저는 특정 언어나 프레임워크보다  
**서비스가 실제 환경에서 어떻게 동작하는지**에 더 큰 관심을 두고 있습니다.

JSP/Servlet, PHP, Node.js 기반 웹 서비스를 직접 구현하며  
서버 배포와 운영을 병행했고,  
VM 환경에서의 리눅스 서버 구성과 데이터베이스 연동,  
OWASP Top 10 기준의 보안 점검을 수행해 왔습니다.

각 프로젝트마다  
- 왜 이 구조가 필요한지  
- 어디에서 장애나 보안 문제가 발생할 수 있는지  
를 중심으로 설계하고 기록해 왔습니다.
